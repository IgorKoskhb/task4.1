# ***Как создать свой проект на Git Hub?***


<font color="gree">

>1. Нажмите на свою иконку
>2. Нажмите на ***YOUR REPOSITORIES***
>3. Нажмите на ***NEW***
>4. Придумайте название своему проекту и напиши это в ***REPOSITORY NAME***
>5. Выберите статус ***PABLIC***
>6. Нажмите на ***CREATE***

</font>
