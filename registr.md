## ***Как зарегистрироваться на Git Hub?***

</font>
 

<font color="black">

>1. Перейдите по ссылке https://github.com/.
>2. Выберите «Sign up».
>3. Заполните информацию.
>4. Нажмите кнопку «Create account».
>5. Войдите в почту, подтвердите почтовый адрес.
